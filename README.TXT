TÍTULO DO PROJETO: Cadastro de Alunos
OBJETIVO DO PROJETO: Implementar um sistema de gerenciamento de alunos aplicando conceitos de Programação Orientada a Objetos em Java. O projeto explora herança (Aluno herdando de Pessoa), realização de interfaces (Armazenador) e manipulação estruturada de arrays para inserir, validar duplicidade, listar e remover registros baseados no RA. 
VERSÃO ou DATA: 1.0 (Referente à atividade de 17/03/2025)
COMO INICIAR O PROJETO: Abra a pasta do projeto no BlueJ. Compile todas as classes. Clique com o botão direito nas classes "TestePessoa" ou "TesteAluno" e selecione o método "void main(String[] args)" para visualizar a execução no terminal.
AUTORES: João Lucas de Lima Souza (Classes estruturais e testes) / Prof. Julio Arakaki (Classe NomePessoa fornecida)
INSTRUÇÕES PARA O USUÁRIO: A aplicação roda puramente via console. Os testes já vêm pré-configurados na classe "TesteAluno" para demonstrar a regra de negócio completa (validação de RA duplicado, remoção com reordenação de array e listagem). Caso deseje testar novos cenários ou cadastrar alunos diferentes, abra o código-fonte da classe "TesteAluno", altere os dados diretamente nas instâncias (nome, idade, RA e semestre), recompile e execute novamente.
